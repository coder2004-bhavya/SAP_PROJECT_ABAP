# SAP_PROJECT_ABAP
SAP ABAP report (ZVX_P2P) for real-time Procure-to-Pay tracking — joins EKKO/EKPO/EKET/MSEG/RBKP/BSEG in a single Open SQL query, auto-classifies PO lines into 5 P2P statuses, and renders a CL_SALV_TABLE ALV grid with drill-down to ME23N/MIR4. SAP ECC 6.0 / S/4HANA 2023 | KIIT SAP CoE Capstone.
